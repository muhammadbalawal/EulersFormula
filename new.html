Mandelbrot Set Explorer

The Mandelbrot set is a fractal that represents a set of complex numbers. Its beauty lies in the intricate patterns created by iterating a mathematical formula repeatedly.

To understand how it works:

  • Zoom: Determines how closely you view the Mandelbrot set.
  • Offset X: Shifts the view horizontally.
  • Offset Y: Shifts the view vertically.
  • Max Iterations: Controls the number of iterations used to determine if a point belongs to the Mandelbrot set. Higher values provide more detailed patterns but increase computation time.

As you adjust these parameters in real-time:

  • Zooming in: Reveals more intricate structures within the fractal.
  • Changing offsets: Lets you explore different areas of the Mandelbrot set.
  • Increasing iterations: Produces smoother transitions and more defined boundaries for the fractal pattern.

The real-time updates make it easy to explore the Mandelbrot set dynamically, revealing its mesmerizing complexity with every tweak!
